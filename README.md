# Akademisk-prakse-II-CAD-faili
Šajā mapē ir iekļauti STL faili šļirces mehāniskās dozēšanas sistēmas
komponentēm, kas izstrādātas un izmantotas akadēmiskās prakses ietvaros.

Komponentes ir pielāgotas 20 ml šļircēm un paredzētas 3D drukai.
Failu nosaukumi atbilst detaļu funkcionālajai lomai sistēmā.
